Global messages can have return value.
Local messages might have restriction that they cannot have return value.

Request template
    Type: 
    Params:
    Returns:
        Status of request

    Type: Global
    Params:
    Returns:
        Status of request

Create new lobby
    Type: Global
    Params:
        Name of lobby
        Password of lobby
    Returns:
        Status of request
        ID of lobby

Join lobby
    Type: Global
    Params:
        ID of lobby
        Password
        Information for websocket connection
    Returns:
        Status of request
        Access token

Leave lobby

    Params:
        ID of lobby
        Access token
    Returns:
        Status of request

Start round
    Type: Local
    Params:
        ID of lobby
        Access token

        Status of request

        ID of lobby
        Access token
        Position of tower (int, int)?
    Returns:
        Status of request

Sell tower
    Type: Local
    Params:
        ID of lobby
        Access token
        ID of tower
    Returns:
        Status of request
